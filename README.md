Este proyecto consiste en un **sistema de inventario básico** desarrollado en Python que permite al usuario **agregar productos y visualizarlos en pantalla** mediante un menú interactivo.  
El programa funciona en consola y utiliza una estructura sencilla, ideal para comprender la gestión básica de datos en listas.

El sistema se mantiene en ejecución hasta que el usuario decide salir, permitiendo realizar múltiples operaciones sin reiniciar el programa. Esto simula el funcionamiento básico de un inventario real en pequeños sistemas administrativos.

Además, el proyecto refuerza la lógica de control de flujo y la interacción con el usuario, mostrando mensajes claros y validando las opciones ingresadas.

#
### 🧑‍💻 Lenguaje de programacion
- Python

#
### 🎯 Objetivos del proyecto
- El objetivo principal del proyecto es **reforzar el uso de listas, bucles y condicionales en Python**. 
- mejorar la comprensión del manejo de menús interactivos en programas de consola.  
- También busca desarrollar la capacidad de organizar información de forma estructurada y dinámica mediante colecciones.

#
### 🧠 Temas que se a aplicado
- Variables
- Listas
- Bucles `while`
- Condicionales `if / elif / else`
- Entrada y salida de datos
- Menús interactivos en consola

#
### ⚙️ Funcionamiento
1. El programa muestra un **menú principal** con varias opciones.
2. El usuario puede:
   - Agregar productos al inventario.
   - Visualizar todos los productos almacenados.
   - Salir del sistema.
3. Los productos se almacenan en una **lista** durante la ejecución del programa.
4. El sistema se repite hasta que el usuario selecciona la opción de salida.
